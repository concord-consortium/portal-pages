Molecular Workbench

Molecular Workbench (MW) is a modeling tool for designing and conducting computational experiments across science. It provides an authoring system for instructional designers to create and publish model and simulation-based curriculum materials, and delivers an interactive learning environment that supports science inquiry. It is also free and open-source!
